Privacy Policy

This Privacy Policy applies to the ChronoVox application. The developers of ChronoVox value and respect users’ privacy. Below is a detailed explanation of how information is handled within the application.

Data Collection

The ChronoVox application does not collect or store any user data. No personal information such as names, surnames, email addresses, contact details, location data, or any other personal information is collected or stored.

The application also does not collect non-personal information, such as usage statistics or device information including device model, operating system, or internet service provider.

Data Sharing

ChronoVox does not share any information with third parties. All information generated within the application is used exclusively for its internal functionality and is not shared with any external entity.

Use of Permissions

ChronoVox may request the following permissions (depending on the Android version in use) strictly for operational purposes. These permissions are not used for any other purpose.

  • Foreground Services and Audio Playback:
    Used to ensure that the timer continues functioning correctly even when the application is not in the foreground or when the device is locked. These permissions allow the use of Text-to-Speech and user-configured alert sounds when the timer finishes.
  • Notifications:
    Used to display the timer status in the notification bar and to alert the user when the timer finishes.
  • Storage:
    Access is requested solely to allow users to save statistic screenshots, export results, and save or load name lists locally on their device. The application does not access, transmit, or synchronize these files with external servers or cloud services.
  • Run at Startup:
    Used exclusively to restore active timers or alerts configured by the user prior to a device restart. The application does not run in the background without prior user action.
  • Vibration:
    Used to provide alerts and confirmations during app usage.

The use of these permissions does not involve the collection, storage, or transmission of user data.

Data Deletion

Since ChronoVox does not collect or store personal data and does not use external servers, there is no user information to delete.

Changes to This Privacy Policy

This Privacy Policy may be updated at any time. Users will be informed of any changes through an application update or other appropriate means.
